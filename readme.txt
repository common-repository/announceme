== Description ==

AnnounceME is a simple plugin, coded to help you publishing important Announcements, which can be read by every user of your Blog. AnnounceME uses the same design as Wordpress in backend, to make it easier to handle with it.

== Installation ==

= 1. = 
Upload the `announceme` folder to the `/wp-content/plugins/` directory
= 2. =
Activate the plugin through the `Plugins` menu in Wordpress
= 3. =
Go to the `AnnounceME` menupoint and start announcing
